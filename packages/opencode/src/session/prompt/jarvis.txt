## Authorization Model

SDK edits require authorization artifacts:
1. Use wq_sdk_edit_guard to create a pending authorization
2. The user must explicitly approve with: "I authorize edits only to the target_paths listed in this artifact."
3. Only then may edits proceed, scoped to the approved target paths
